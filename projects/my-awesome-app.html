My Awesome App

A modern social media app built with Kotlin and Jetpack Compose.

Overview

My Awesome App enables users to create posts, follow friends, and engage in real-time conversations. Built with Jetpack Compose, it offers a fluid, responsive UI and leverages Kotlin coroutines for efficient async operations.

Architecture

The app follows MVVM with a clean architecture approach: data, domain, and presentation layers. It uses Room for local persistence, Retrofit for network calls, and Hilt for dependency injection.

Highlights

  • Jetpack Compose UI with Material 3
  • Offline-first with Room and caching
  • Coroutines + Flow for reactive streams
  • Accessibility and performance auditing
